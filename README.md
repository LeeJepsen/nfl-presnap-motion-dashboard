# NFL Pre-Snap Motion Dashboard

Interactive Streamlit dashboard built using NFL Big Data Bowl 2025 tracking data.

## Tab 1 – League Overview

- Week range filtering
- League motion KPIs
- Team motion comparison
- Defensive adjustment modeling

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app/app.py

## Data
Derived from NFL Big Data Bowl 2025 (Weeks 1–9).
```
